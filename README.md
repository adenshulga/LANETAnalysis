Как использовать. Инструкция

Извлечение данных из модели оставляется за пользователем, так как код моделей и внутренний формат данных обычно разный.

Вы должны унаследоваться от класса ProcessedDataset из файла utils/DataClass.py и обязательно переопределить в нем абстрактные методы родительского класса, а также поля model_name и dataset_name. Каждый метод должен возвращать np.ndarray размера (num_of_samples, num_of_labels)

Обьект вашего заданного подкласса передаете на вход инизиализатору класса DataClass.

Далее, пример использования полученного обьекта можно прочитать в ноутбуке example.ipynb
